## 4. 画面設計

### 4.1 未締めアラートの判定・表示

**判定ロジック**
```
アクティブな会計年度ごとに:
  前月以前の 08_getsukikanryo で status = 'pending' のレコードを検索
  → 存在する場合アラート表示

表示箇所:
  ① ナビバー: 警告バッジ（未締め件数）
  ② ホーム画面: バナー「⚠ ○○ ○月分が未締めです」
```
