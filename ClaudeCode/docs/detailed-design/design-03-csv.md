## 3. CSVフォーマット仕様

### 3.1 予算科目CSV（年度初めにインポート）

#### 歳入科目CSV（確定）

列構成: `所管,款,項,目,予算額,説明`

```csv
所管,款,項,目,予算額,説明
山田太郎,所得収入,給与,基本給,4000000,会社からの月次給与
山田太郎,所得収入,給与,超勤手当,,
山田太郎,所得収入,給与,住宅手当,,
山田太郎,所得収入,給与,国内駐在手当,,
山田太郎,所得収入,給与,その他手当,,
山田太郎,所得収入,賞与,基本分,,
山田太郎,所得収入,賞与,評価反映分,,
山田太郎,所得収入,賞与,DCキャッシュ,,
山田太郎,所得収入,その他,その他手当,,
山田太郎,贈与収入,贈与,じいじ,,
山田太郎,贈与収入,贈与,ばあば,,
山田太郎,贈与収入,贈与,,0,贈与収入なし
山田太郎,他会計受入,住居移転関連基金受入,,,住居移転関連基金からの繰入
山田太郎,他会計受入,ふるさと納税基金受入,,,
山田太郎,他会計受入,財政調整基金受入,,,
山田太郎,立替返金,旅行関係費立替返金,旅行関係費,,
山田太郎,立替返金,交際費立替返金,交際費,,
山田太郎,立替返金,会社交通費立替返金,会社交通費,,
山田太郎,立替返金,,,500000,立替返金全体の概算
```

| フィールド | 必須 | 説明 |
|---|---|---|
| 所管 | ○ | ユーザー名（`01_user.name` と照合）→ `shokan_code` を取得 |
| 款 | ○ | 款名（07_kyotsuteisuteigi 区分03で検索）→ 款コード（KK）を取得 |
| 項 | △ | 項名（07_kyotsuteisuteigi 区分04で検索）。**空白の場合は款レベル参考値行**として処理 |
| 目 | △ | 目名（07_kyotsuteisuteigi 区分05で検索）。**空白の場合は目なし**（`MMM=000`）として処理 |
| 予算額 | △ | 予算額。**空白=NULL**（後で画面設定）、**0=0円**として保存 |
| 説明 | △ | 科目の説明・備考。**空白=NULL**。`05_yosankamoku.description` に保存 |

**インポート時の自動採番フロー**:
```
[パターン1: 通常（目あり・予算額あり・説明あり）]
[CSV行] 山田太郎, 所得収入, 給与, 基本給, 4000000, 会社からの月次給与
    ↓ 07_kyotsuteisuteigi 検索
  shokan_code: "01" / KK: "01" / KO: "01" / MMM: "001"
    ↓ 05_yosankamoku に生成（level=目）
  code: "01-01-001", level: "目", description: "会社からの月次給与"
    ↓ 04_yosanhaibun に生成
  amount: 4000000

[パターン2: 目なし・予算額0]
[CSV行] 山田太郎, 贈与収入, 贈与, , 0
    ↓ 目=空白 → MMM=000
  code: "02-04-000", level: "項"  ← 目レコードは作成しない
    ↓ 04_yosanhaibun に生成
  amount: 0

[パターン3: 予算額未設定（NULL）]
[CSV行] 山田太郎, 立替返金, 交際費立替返金, 交際費,
    ↓ 通常処理（level=目）
  code: "04-09-014", level: "目"
    ↓ 04_yosanhaibun に生成
  amount: NULL  ← 後で画面から設定

[パターン4: 款レベル参考値]
[CSV行] 山田太郎, 立替返金, , , 500000
    ↓ 項・目=空白 → 款レベル行
  code: "04-00-000", level: "款"  ← 既存の款レコードを使用
    ↓ 04_yosanhaibun に生成
  amount: 500000  ← 参考値（集計・執行率の対象外）
```

**その他の処理**:
- 親子関係（parent_id）は款→項→目の順で自動設定
- 同一CSVで款・項・目レコードを階層順に一括生成
- 暫定予算フラグ（is_provisional_target）はインポート後に画面から設定

#### 歳出科目CSV（フォーマット案）

> ※ 歳出科目のCSVフォーマットは別途確定する。歳入と同様に `所管,款,項,目` 形式を想定。

### 3.2 実績CSV（月次インポート）

```csv
date,amount,direction,category
2026-01-25,264000,歳入,給与
2026-01-25,23790,歳出,年金保険料
2026-01-05,3500,歳出,食費
2026-01-07,12000,歳出,交通費
```

| フィールド | 必須 | 説明 |
|---|---|---|
| date | ○ | 収支日（YYYY-MM-DD） |
| amount | ○ | 金額（正の整数） |
| direction | ○ | `歳入` or `歳出` |
| category | △ | 機械的な分類（誤りを含む場合あり） |
