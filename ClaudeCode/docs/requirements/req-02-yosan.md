## 4. 予算管理モジュール

### 4.1 予算科目体系

#### 4.1.1 歳入

> **款・項・目の設計思想（本システムの定義）**:
> - **款** = 性質（何の種類の収入か）
> - **項** = 収入源（どこから入ってくるか）
> - **目** = 内訳（具体的に何が入ってきたか）

```
コード形式: SS-KK-KO-MMM
  SS  = 所管コード（2桁）   例: 01=山田太郎
  KK  = 款コード（2桁）     例: 01=所得収入
  KO  = 項コード（2桁）     例: 01=給与
  MMM = 目コード（3桁）     例: 001=基本給 / 000=目なし

完全なコード例: 01-01-01-001（所得収入>給与>基本給）
```

#### 4.1.2 歳出

> **目・事項の設計思想（本システムの定義）**:
> - **項** = 費目（何の種類の支出か）
> - **目** = 細目（具体的に何に使ったか）
> - **事項** = 目的（何のために支出したか）
>
> 目と事項に主従関係はない。科目別帳票・事項別帳票の2形式でそれぞれ独立して集計する。
> 例: 旅行関係費（項）＞交通費（目）に対し、社会科見学・ホノルルマラソン等（事項）が紐付く。

```
コード形式: SS-KK-MMM
  SS  = 所管コード（2桁）   例: 01=山田太郎
  KK  = 項コード（2桁）     例: 01=控除
  MMM = 目コード（3桁）     例: 001=所得税 / 000=目なし

完全なコード例: 01-01-001（控除>所得税）
※ 事項はコードによる管理は行わない。名称（テキスト）で管理し、行事（10_gyojikeikaku）との紐付けに使用する。
```

**特殊科目の処理方針（行政レビュー確定 / 2026-02-28）**

| 科目 | 処理方針 | 根拠 |
|---|---|---|
| 控除（給与天引き） | 法令に基づく**強制的支出**として歳出計上する。毎月自動的に発生する定額支出として取り扱う | 予算単一の原則に準拠。歳出と別建てすることは不可 |
| 立替金 | **総計予算主義**の原則に基づき一旦歳出計上する。精算回収時は「立替返金」歳入（04_立替返金款）に計上する | 歳入歳出は相殺せず全額を計上する原則に準拠 |
| 将来への備え＞自社株 | 残高管理が不可能なため**消費的歳出**として計上する。払い戻し（売却・配当等）があった場合は「雑収入」等の歳入に計上する | 残高把握不可という現実的制約による例外的取り扱い |
| ふるさと納税基金繰入 | ふるさと納税は**暦年（1/1〜12/31）**ベースのため**特別会計**で管理する。資金フロー: 一般会計→ふるさと納税基金（積立）→ふるさと納税特別会計（取崩・歳出） | 一般会計の年度（4/1〜3/31）と対象期間が異なるため特別会計に分離 |

**特別会計の歳入科目例**
```
基金受入 ← 方式B: 基金取崩時の受入科目
```

### 4.2 予算科目管理

| 項目 | 内容 |
|---|---|
| 登録方法 | 年度初めにCSVファイルからインポート |
| インポート後 | 手動での個別追加・編集も可能 |
| 暫定予算フラグ | 科目単位で `is_provisional_target` フラグを設定（インポート後に画面から設定） |

### 4.3 実績データ管理

#### インポート仕様

| 項目 | 内容 |
|---|---|
| 登録方法 | 月次でCSVファイルからインポート |
| 対象 | 歳入・歳出の両方 |
| CSVの項目 | 日付、金額、分類（機械的分類のため誤りを含む場合あり） |

**実績CSVフォーマット**
```csv
日付,項目,分類,金額,表示
2026-01-05,給与振込,給与,264000,対象
2026-01-05,スーパーで買い物,食費,3500,対象
2026-01-07,電車賃,交通費,12000,対象
2026-01-10,Suicaチャージ,交通費,10000,対象外
```

#### 割り振り画面（UI要件）

インポートした実績データを予算科目に紐付ける専用画面。

```
┌──────────────────────────────┬────────────────────────────────┐
│  未割当の実績（左ペイン）      │  予算科目ツリー（右ペイン）      │
│                              │                                │
│ フィルタ: [未割当▼] [月選択▼] │  ▼ 02 歳出                     │
│ 歳入/歳出: [全件▼]           │    ▼ 02-04 固定費               │
│                              │      ▼ 02-04-02 食費            │
│ □ 01/05 給与    264,000円    │        ┌──────────────────────┐ │
│ □ 01/05 食費      3,500円    │        │ 02-04-02-01 自炊費   │ │
│ □ 01/07 交通費  12,000円     │        │ 予算:  60,000円      │ │
│                              │        │ 実績:   3,500円      │ │
│  複数選択 → D&D で割り振り    │        │ 残額:  56,500円      │ │
│                              │        └──────────────────────┘ │
└──────────────────────────────┴────────────────────────────────┘
```

**操作仕様**
- 左ペインで実績データを複数選択（チェックボックス）
- 右ペインの科目箱へドラッグ＆ドロップで割り振り
- 一度割り振った実績の再割り振りも可能
- 左ペインはフィルタ（未割当／割当済／全件）・月・歳入歳出区分で絞り込み可能
- 科目箱には予算額・実績累計・残額をリアルタイム表示
- 割当先がない実績(例：二重計上)も存在する可能性があるため、削除フラグを設けて完全削除は行わない運用とする
