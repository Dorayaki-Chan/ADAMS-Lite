## 4. 予算管理モジュール

### 4.1 予算科目体系

**所管（ユーザー）> 款（大分類）> 項（中分類）> 目（小分類）の4階層構造**。

- **所管**: `05_yosankamoku.shokan_code`（07_kyotsuteisuteigi 区分02）で管理。CSVには「所管」列として記載。
- **款・項・目**: `05_yosankamoku.code`（`KK-KO-MMM` 形式）で 07_kyotsuteisuteigi を参照して自動採番。
- **会計年度**: `05_yosankamoku.fiscal_year_code`（07_kyotsuteisuteigi 区分01）で管理。
- **歳入・歳出の区別**: `05_yosankamoku.direction`（`revenue` / `expenditure`）で管理。

```
コード形式: KK-KO-MMM
  KK  = 款コード（2桁）  例: 01=所得収入
  KO  = 項コード（2桁）  例: 01=給与
  MMM = 目コード（3桁）  例: 001=基本給 / 000=目なし

完全なコード例: 01-01-001（所得収入>給与>基本給）
```

**歳入科目の分類（令和8年度暫定）**
```
所得収入
  └─ 給与
  │    ├─ 基本給
  │    ├─ 超勤手当
  │    ├─ 住宅手当
  │    ├─ 国内駐在手当
  │    └─ その他手当
  └─ 賞与
  │    ├─ 基本分
  │    ├─ 評価反映分
  │    └─ DCキャッシュ
  └─ その他
       └─ その他手当
贈与収入
  └─ 贈与
       ├─ じいじ
       └─ ばあば
他会計受入（※目なし：項が最下位）
  └─ 住居移転関連基金受入
  └─ ふるさと納税基金受入
  └─ 財政調整基金受入
立替返金
  └─ 旅行関係費立替返金
  │    └─ 旅行関係費
  └─ 交際費立替返金
  │    └─ 交際費
  └─ 会社交通費立替返金
       └─ 会社交通費
```

**歳出科目（部: 02）の例（一般会計）**
```
控除（税金・保険料）
趣味（旅費・設備投資）
変動費（日用品・交際・医療・被服・美容・税金等・予備費）
固定費（住宅・食費・水道光熱費・通信費・新聞費）
政策推進費（詳細確認中）
基金積立金 ← 方式B: 基金への積立を一般会計の歳出として管理
  └─ ふるさと納税基金積立
  └─ 住居移転関連基金積立
```

**特別会計の歳入科目例**
```
基金繰入収入 ← 方式B: 基金取崩時の受入科目
```

### 4.2 予算科目管理

| 項目 | 内容 |
|---|---|
| 登録方法 | 年度初めにCSVファイルからインポート |
| インポート後 | 手動での個別追加・編集も可能 |
| 暫定予算フラグ | 科目単位で `is_provisional_target` フラグを設定（インポート後に画面から設定） |
| 予算額 | **CSVの第5列から設定可能**。空白=NULL（後で画面設定）、0=0円で登録。款レベルは参考値（集計対象外） |

**歳入予算科目CSVフォーマット（確定）**

列構成: `所管,款,項,目,予算額,説明`

```csv
所管,款,項,目,予算額,説明
山田太郎,所得収入,給与,基本給,,
山田太郎,所得収入,給与,超勤手当,,
山田太郎,所得収入,給与,住宅手当,,
山田太郎,所得収入,給与,国内駐在手当,,
山田太郎,所得収入,給与,その他手当,,
山田太郎,所得収入,賞与,基本分,,
山田太郎,所得収入,賞与,評価反映分,,
山田太郎,所得収入,賞与,DCキャッシュ,,
山田太郎,所得収入,その他,その他手当,,
山田太郎,贈与収入,贈与,じいじ,,
山田太郎,贈与収入,贈与,ばあば,,
山田太郎,他会計受入,住居移転関連基金受入,,,
山田太郎,他会計受入,ふるさと納税基金受入,,,
山田太郎,他会計受入,財政調整基金受入,,,
山田太郎,立替返金,旅行関係費立替返金,旅行関係費,,
山田太郎,立替返金,交際費立替返金,交際費,,
山田太郎,立替返金,会社交通費立替返金,会社交通費,,
```

- **所管**: システムの `01_user.name` と照合してインポート対象ユーザーを特定
- **目列が空白の場合**: 目なし扱い（項レベルが最下位。目レベルのレコードは作成しない）
- **項・目がともに空白の場合**: 款レベルの参考値行。予算額を款に紐付けて保存するが集計対象外
- **予算額が空白の場合**: `04_yosanhaibun.amount = NULL` として登録（後で画面から設定）
- **予算額が 0 の場合**: 明示的な0円として登録
- **説明が空白の場合**: `05_yosankamoku.description = NULL` として登録
- **コード**: インポート時にシステムが自動採番

**歳出予算科目CSVフォーマット（案）**

> ※ 歳出科目のCSVフォーマットは別途確定する。

### 4.3 実績データ管理

#### インポート仕様

| 項目 | 内容 |
|---|---|
| 登録方法 | 月次でCSVファイルからインポート |
| 対象 | 歳入・歳出の両方 |
| CSVの項目 | 日付、金額、分類（機械的分類のため誤りを含む場合あり） |

**実績CSVフォーマット（案）**
```csv
date,amount,direction,category
2026-01-05,264000,歳入,給与
2026-01-05,3500,歳出,食費
2026-01-07,12000,歳出,交通費
```

#### 割り振り画面（UI要件）

インポートした実績データを予算科目に紐付ける専用画面。

```
┌──────────────────────────────┬────────────────────────────────┐
│  未割当の実績（左ペイン）      │  予算科目ツリー（右ペイン）      │
│                              │                                │
│ フィルタ: [未割当▼] [月選択▼] │  ▼ 02 歳出                     │
│ 歳入/歳出: [全件▼]           │    ▼ 02-04 固定費               │
│                              │      ▼ 02-04-02 食費            │
│ □ 01/05 給与    264,000円    │        ┌──────────────────────┐ │
│ □ 01/05 食費      3,500円    │        │ 02-04-02-01 自炊費   │ │
│ □ 01/07 交通費  12,000円     │        │ 予算:  60,000円      │ │
│                              │        │ 実績:   3,500円      │ │
│  複数選択 → D&D で割り振り    │        │ 残額:  56,500円      │ │
│                              │        └──────────────────────┘ │
└──────────────────────────────┴────────────────────────────────┘
```

**操作仕様**
- 左ペインで実績データを複数選択（チェックボックス）
- 右ペインの科目箱へドラッグ＆ドロップで割り振り
- 一度割り振った実績の再割り振りも可能
- 左ペインはフィルタ（未割当／割当済／全件）・月・歳入歳出区分で絞り込み可能
- 科目箱には予算額・実績累計・残額をリアルタイム表示
