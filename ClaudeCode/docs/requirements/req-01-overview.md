## 1. システム概要

官庁会計の考え方（予算執行・決算）を家計に適用し、旅費管理も含めた一元的な家計管理システムを構築する。

| モジュール | 概要 |
|---|---|
| 予算管理 | 予算科目の設定・実績データの割り振り・追跡 |
| 決算管理 | 月次・年度締め処理・報告書生成 |
| 行事計画 | 旅行等の大規模・複数月跨ぎ支出の計画管理・予算仮押さえ |
| 旅費精算 | 実績割り振り画面に統合（行事フィルタで対応） |
| 現金精算即時連携 | Discord経由で外出先の現金支払いを即時登録 |

| 項目 | 内容 |

## 3. 会計制度

### 3.1 会計区分

| 区分 | 数 | 予算編成 | 備考 |
|---|---|---|---|
| 一般会計 | 1つのみ | あり | 4月〜翌年3月の年度管理 |
| 特別会計 | 複数・変動 | あり | 目的別（例: ふるさと納税特別会計） |
| 基金 | 複数・変動 | **なし** | 積立金。残高管理のみ（例: 住居移転関連基金） |

**基金の特性（方式B統一）**
- 予算科目との紐付けなし
- 積立（入金）・取崩（出金）で残高を管理
- **積立時**: 歳出（基金繰入）として支出し、基金残高を増やす
- **取崩時**: 基金残高を減らし、歳入（基金受入）として受け入れる

```
【積立時】
  一般会計 歳出: 基金繰入（02-XX-XX） → 基金残高 +600,000円

【取崩・執行時】
  基金残高 -600,000円 → 特別会計 歳入: 基金受入 +600,000円
                         特別会計 歳出: 目的支出     -600,000円

例: 令和７年度一般会計 → 住居移転関連基金 → 令和８年度一般会計
  ① 令和７年度一般会計 歳出「基金繰入」で基金に積立
  ② 住居移転時に基金取崩 → 令和８年度一般会計 歳入「基金受入」へ受入
  ③ 令和８年度一般会計 歳出「住居移転費用」で執行
```

### 3.2 年度期間

| 区分 | 開始 | 終了 | 年度締め |
|---|---|---|---|
| 一般会計 | 4月1日 | 翌年3月31日 | 翌年度5月下旬〜6月上旬 |
| 特別会計 | 任意 | 任意 | 任意 |

### 3.3 予算の形態

| 形態 | 発生タイミング | 特性 |
|---|---|---|
| 当初予算（本予算） | 年度開始前 | 必ず存在する |
| 補正予算 | 年度中（12月頃） | **積み上げ方式**（当初予算に加減算）。減額も発生する |
| 暫定予算 | 4月（本予算未成立時） | 本予算成立後に廃止・本予算へ移行 |

**補正予算の積み上げ方式**
```
当初予算額   + 補正予算第1号  + 補正予算第2号  = 現行予算額（実績と比較）
例: 食費  900,000円  -101,000円（減額）               799,000円
```

**暫定予算の計算式**
```
対象科目: 生活共通費など生活上必要最低限の費目（is_provisional_target フラグで識別）
計算式: 前年度実績合計 × (暫定期間の月数 ÷ 12)
例: 前年度食費 720,000円 × 2ヶ月 ÷ 12 = 120,000円
```
