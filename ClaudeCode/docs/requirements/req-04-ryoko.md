## 6. 旅行計画モジュール（行事計画）

> **ステータス**: 要件定義中（DB設計・画面設計は歳出科目体系確定後に実施）

### 6.1 背景・課題

旅行などの行事は、**実施日と支払日が大きくズレる**ことが多く、通常の予算実績管理では今年度の残額が不透明になる。

| 支払内容 | 支払日 | 実施日 |
|---|---|---|
| 飛行機予約 | 12/10 | 1/13 |
| ホテル予約 | 1/5 | 1/13〜1/15 |
| 現地経費 | 1/13〜1/15 | 1/13〜1/15 |

→ 旅行関係費が**今年度いくら残っているか常に不透明**という課題がある。

### 6.2 対象範囲

旅行に限らず、**支出を伴う行事・目的全般**を対象とする。

**行事の種類（type）**

| type | 概要 | 例 |
|---|---|---|
| `event` | 日程が決まっている大規模行事。期間（start_date / end_date）を持つ | 社会科見学・ホノルルマラソン・旅行 |
| `purpose` | 繰り返し発生する小目的。日程によらず実績に「目的」ラベルとして付与する | 弱男定例会・個人交際 |

### 6.3 解決アプローチ

行事ごとに事前に**予算計画**を立て、関連支出を行事に紐付けることで、予算残額を「行事仮押さえ分」として可視化する。

### 6.4 機能要件

**行事管理**
- 行事の登録（名称・カテゴリ・期間・対象予算科目・予定総額）
- 行事のステータス管理（計画中 / 仮押さえ中 / 実行中 / 完了）

**支払計画管理**
- 行事に紐付く支払予定の登録（予定日・金額・内容・支払先）
- 支払実績（`06_shiharaijisseki`）との紐付け（実際の支払い発生時）

**予算の仮押さえ**
- 行事の予定総額を対象予算科目から仮押さえ
- 仮押さえ中は予算残額画面に「（行事仮押さえ: △△円）」として別表示
- 実績紐付け完了時に仮押さえを解除し、確定実績として計上

**行事予算配分管理（目 × 事項）**
- 行事登録後、「どの予算科目（目）にいくら使うか」を目単位で入力（→ `11_gyojiyosanhaibun`）
- 1行事 × 複数目 の組み合わせで予算額を管理
- 例: 社会科見学（event） × 旅行関係費＞交通費 = 15,000円 / 宿泊費 = 30,000円 / 食費 = 8,000円
- 入力した事項別予算は帳票（科目別・事項別の2形式）に反映される

> **【積算方式 / 案A採用】** 目の予算額 = 当該目に紐付く全事項の合計（`SUM(11_gyojiyosanhaibun.amount)`）。
> `04_yosanhaibun` への直接登録は不要（国の予算書と同様、事項合計 = 目合計 = 項合計）。
> 例: 旅行関係費＞交通費の予算 = 社会科見学15,000 + ホノルルマラソン80,000 + 佐世保旅行25,000 = **120,000円**（自動計算）

**表示**
- 行事別収支状況（予定額 / 確定済み / 残り支払予定）
- ホーム画面に進行中行事の一覧ウィジェット表示

**事項の種別（確定設計）**

| 種別 | 概要 | 例 |
|---|---|---|
| **説明型** | 目の用途を説明するテキストラベル。`10_gyojikeikaku` とは連携しない | 「所得税の納付に要する費用」「住居の維持に要する費用」 |
| **行事型** | `10_gyojikeikaku` に登録された行事と紐付く事項。`11_gyojiyosanhaibun` を通じて予算管理される | 「社会科見学」「ホノルルマラソン」「佐世保旅行」 |

> 基本は**説明型**として扱う。行事計画作成画面で、どの事項と紐づけるかを選択することで行事型に昇格する。

**行事計画作成時の変更区分**

| 変更区分 | 動作 |
|---|---|
| **なし** | 既存の事項に行事をそのまま紐付ける（事項予算額は変更しない） |
| **分割** | 1つの事項（説明型）を複数の行事に分けて充てる |
| **流用** | 別の用途で設定されていた事項の予算を、この行事に充てる |
| **新規作成** | 対応する事項が存在しない場合に、新規で事項を作成して紐付ける（事項列は非活性） |

**4パターンの動作例**

| パターン | 変更区分 | 状況 |
|---|---|---|
| ① | なし | 事項「福岡旅行に要する費用」に行事「福岡旅行」をそのまま紐付け |
| ② | 分割 | 事項「スキー旅行に要する費用」を「ハンタマスキー場」と「苗場スキー場」の2行事に分割 |
| ③ | 流用 | 事項「北海道旅行に要する費用」を、行き先変更で行事「沖縄旅行」に流用 |
| ④ | 新規作成 | 旅行関係費の予算余剰を使い、事項のない行事「仙台旅行」を新規登録 |

### 6.5 帳票との関係

国の予算書（甲号）に倣い、以下の2形式の帳票出力を目指す。

| 帳票形式 | 政府予算書の対応 | 行単位 | 概要 |
|---|---|---|---|
| **科目別帳票** | 歳入歳出予算各目明細書（1091形式） | 目ごとに事項を展開 | 項 → 目 → 事項① → 事項② の順で縦展開 |
| **事項別帳票** | 歳入歳出予算事項別明細書（1085形式） | 事項ごとに目を展開 | 事項（行事）ごとに使用した目と金額を一覧表示 |

**科目別帳票のイメージ（旅行関係費の部分）**:
```
 項: 旅行関係費
   目: 交通費          予算 120,000円  実績  15,000円  残 105,000円
     事項① 社会科見学          予算  15,000円  実績  15,000円  残      0円
     事項② ホノルルマラソン    予算  80,000円  実績       0円  残  80,000円
     事項③ 佐世保旅行          予算  25,000円  実績       0円  残  25,000円
   目: 宿泊費          予算 200,000円  実績       0円  残 200,000円
     事項① 社会科見学          予算  30,000円  実績       0円  残  30,000円
     事項② ホノルルマラソン    予算 120,000円  実績       0円  残 120,000円
```

**事項別帳票のイメージ（社会科見学）**:
```
 事項: 社会科見学
   旅行関係費 ＞ 交通費    予算  15,000円  実績  15,000円  残      0円
   旅行関係費 ＞ 宿泊費    予算  30,000円  実績       0円  残  30,000円
   旅行関係費 ＞ 食費      予算   8,000円  実績       0円  残   8,000円
   旅行関係費 ＞ 入館料    予算   5,000円  実績       0円  残   5,000円
   合計                    予算  58,000円  実績  15,000円  残  43,000円
```
