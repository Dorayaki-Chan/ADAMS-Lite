# ADAMS Lite

## 概要
我が家では財政原則に基づいた家計運用を行っています。
予算執行、決算といった官庁会計システムの対象業務に加えて、旅行計画、旅費精算といった旅費システムが対象とする業務機能を導入し、会計業務を一元的に行えるシステムを構築することを目指しています。

## システム構成
ADAMS Liteは、以下のモジュールで構成されています。
- 予算管理モジュール: 予算の設定、管理、実績の追跡を行います。
- 決算管理モジュール: 決算処理、報告書の生成を行います。
- 旅行計画モジュール: 旅行の計画、予算の設定を行います。
- 旅費精算モジュール: 旅費の精算、報告書の生成を行います。

## 使用技術
- フロントエンド: React
- バックエンド: Node.js(TypeScript), Express(TypeScript)
- データベース: MySQL

## 基本設計
ADAMS Liteは、モジュールごとに独立したサービスとして設計されており、APIを通じて通信します。これにより、各モジュールの開発と保守が容易になります。また、データベースは共通のスキーマを使用し、各モジュールが必要なデータにアクセスできるように設計されています。

### ページ構成
- ホームページ: システムの概要と最新の予算・決算情報を表示します。
- 予算管理ページ: 予算の設定、管理、実績の追跡を行います。
- 決算管理ページ: 決算処理、報告書の生成を行います。
- 旅行計画ページ: 旅行の計画、予算の設定を行います。
- 旅費精算ページ: 旅費の精算、報告書の生成を行います。

### API設計
- 予算管理API: 予算の作成、更新、削除、取得を行います。
- 決算管理API: 決算の作成、更新、削除、取得を行います。
- 旅行計画API: 旅行計画の作成、更新、削除、取得を行います。
- 旅費精算API: 旅費精算の作成、更新、削除、取得を行います。

## UI/UX設計
- シンプルで直感的なインターフェースを提供し、ユーザーが簡単に操作できるようにします。
- レスポンシブデザインを採用し、様々なデバイスで快適に利用できるようにします。
- ユーザーの操作に対して適切なフィードバックを提供し、エラーが発生した場合にはわかりやすいエラーメッセージを表示します。
